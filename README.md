# Node-App
"Liri" Node.js App
This is a Node application using the spotify, twitter, and request (resquesting OMDB's API) Node packages. It accepts five commands:
		do-what-it-says
		add-command
		spotify-this-song
		movie-this
		tweets
"tweets=> takes a user input and searches Twitter for it",
"spotify-this-song=> takes a user input and searches songs/artists related to the term",
"movie-this=> takes a user input and searches OMDB api for it",
"do-what-it-says=> runs a random command/search from a csv file",
"add-command=> add a command/search to the random csv file"
